# Projeto de simulação de carteiras digitais

## Funcionalidades
- Criação de usuário
- Busca de usuário por ID
- Transferência entre carteiras digitais
- Agendamento de transferência

## Tecnologias utilizadas
- .NET 5.0
- Web API
- Worker Service
- Service Bus
- CosmosDB (SQL API)
- AKS (Kubernetes)
- Docker
- Terraform (Infra as code)
